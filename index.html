How to use the app

  1. Install the metamask plugin and create an account on Ropsten Test Network and load some Ether. DM @zastrinlab if you need test ether.
  2. Purchase tokens below by entering the total number of tokens you like to buy.
  3. Vote for candidates by entering their name and no. of tokens to vote with.
  4. Enter your account address to look up your voting activity.
